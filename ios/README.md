# Tesil Media — iOS app

Native SwiftUI client for the Tesil Media streaming site. Lives in the same
repo as the Next.js backend so they stay in lockstep.

## Requirements

- macOS with **Xcode 15.3+**
- **iOS 17** simulator or device (uses `@Observable`)
- **XcodeGen** (`brew install xcodegen`) to regenerate the Xcode project from
  [`project.yml`](project.yml)

The `.xcodeproj` is gitignored — `project.yml` is the source of truth so a
new Swift file doesn't produce a 200-line diff in a binary plist.

## First-time setup (Mac)

```bash
cd ios
xcodegen generate
open TesilMedia.xcodeproj
```

In Xcode, set your signing team on the `TesilMedia` target (Signing &
Capabilities tab) before running on a device. Simulator runs need no team.

## Pointing at the backend

The API base URL is read from the `TESIL_API_BASE_URL` build setting (falls
back to `http://localhost:3000` for simulator dev).

To run against your local Next.js dev server from the simulator: nothing to
do — `localhost:3000` works because simulator and Mac share the loopback.

To run against your local server from a **physical device** on the same
Wi-Fi: set `TESIL_API_BASE_URL` in the scheme's environment variables to
`http://<your-mac-LAN-ip>:3000`. You'll also need to relax ATS for that host
in `Resources/Info.plist`.

To run against staging/production: set `TESIL_API_BASE_URL` to the HTTPS URL.

## Project layout

```
TesilMedia/
  TesilMediaApp.swift     — App entry, root view
  Auth/
    Keychain.swift        — Keychain wrapper for tokens
    AuthStore.swift       — @Observable auth state
  Networking/
    APIClient.swift       — URLSession + Bearer + 401-retry
    APIError.swift        — Typed error
    Endpoint.swift        — Path / method / body / auth requirement
  Models/
    Video.swift, User.swift, Channel.swift  — Codable DTOs
  Views/
    RootView.swift        — Switches on auth state
    Auth/                 — SignInView, SignUpView
    Home/                 — HomeFeedView, VideoRow
    Watch/                — WatchView (AVPlayer)
  Resources/
    Info.plist            — generated by XcodeGen from project.yml
```
